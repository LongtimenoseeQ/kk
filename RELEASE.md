# 3.3.0 - 2022/03/17

## 概述

涂鸦 TuyaOS-Zigbee 3.3.0 版本：

​		V3.3.0 版本同时支持 **EFR32MG21F1024IM32** 和 **EFR32MG21F768IM32** 两款芯片，并支持全功能的 Router 设备开发和低功耗设备开发。

​		注意：目前低功耗设备不支持网络自恢复功能，需要在 Zigbee 初始化完成后手动调用 **tal_zg_nwk_recovery_disable(TRUE)** 来禁止网络自恢复功能。
